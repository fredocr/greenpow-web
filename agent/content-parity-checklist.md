# Content Parity Checklist

Use this checklist before adding or changing public pages.

## Route Parity

- English route exists.
- Spanish route exists.
- Routes use the approved URL structure.
- Language switcher points to the equivalent page.

## Layout Parity

- Same template.
- Same component order.
- Same CTA structure.
- Same form behavior.
- Same FAQ structure where applicable.

## Metadata Parity

- English title and description exist.
- Spanish title and description exist.
- Open Graph data exists for both languages.
- Canonical and alternate language links are correct.

## Content Source

- Copy comes from `context/` or approved implementation content.
- Strategic claims match `PROJECT_BRIEF.md`.
- RND content has been promoted before use.
- Legal content has not been improvised inside components.
- Press and media copy comes from `context/press/`.

## Forms

- Form submits through the central handler.
- `preferred_language` is set.
- `source_page` is set.
- UTM fields are preserved when available.
- Success and error messages are localized.
