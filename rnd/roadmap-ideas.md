# Roadmap Ideas

Use this file for future product and website ideas that are not yet approved for implementation.

- Public cloud region recommender.
- Scope 2 infrastructure visibility dashboard.
- Hosting migration estimator.
- Carbon-aware AI batch processing.
- Cortex organizational memory package.
- Partner portal for hosting and infrastructure providers.

## Website Ideas

- Interactive cloud carbon comparison.
- MySx explainer diagram.
- Cortex use-case demo.
- ESG and CSRD readiness assessment.
- Partner qualification flow.
- Investor data room request flow.

## Evaluation Criteria

- Does it support the core positioning?
- Does it help sales, partners, investors, or education?
- Does it improve product-led perception?
- Can it be explained in both English and Spanish?
- Can it be implemented without unsupported impact claims?
