# Cortex Research

## Working Definition

Cortex is a static organizational memory layer for autonomous agents.

It should help agents access stable context, institutional knowledge, process memory, and decision-support structures.

## Research Questions

- What kinds of organizational knowledge should Cortex store?
- How should Cortex differ from a generic document repository?
- How should Cortex support autonomous-agent workflows?
- Which industries need Cortex first?
- What examples make Cortex understandable without overpromising?

## Candidate Knowledge Types

- Company facts.
- Team responsibilities.
- Operating procedures.
- Decision records.
- Sales positioning.
- Compliance boundaries.
- Product context.
- Customer support patterns.
