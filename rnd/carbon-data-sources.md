# Carbon Data Sources

Use this file to track possible data sources for carbon intensity, energy pricing, cloud-region emissions, and infrastructure impact.

## Source Categories

- Grid carbon intensity APIs.
- Energy market price feeds.
- Cloud provider sustainability data.
- Data center energy mix data.
- Regional emissions factors.
- Lifecycle assessment references.

## Evaluation Criteria

- Geographic coverage.
- Update frequency.
- API reliability.
- Licensing terms.
- Historical data availability.
- Fit with MySx decision logic.
- Suitability for public claims.

## Claim Discipline

Do not publish carbon reduction claims until the source, method, and assumptions have been reviewed.
