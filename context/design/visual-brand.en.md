# Visual Brand Context

## Colors

Corporate colors:

- Green: `#8BC34A`
- Blue: `#03A9F4`

Support colors:

- Dark gray: `#303030`
- Light blue: `#B3E5FC`
- Medium blue: `#0288D1`
- Gray: `#727272`
- Light gray: `#B6B6B6`

## Typography

Preferred typography from style guide:

- Avenir Book.
- Avenir Medium.

Fallback web font stack:

```css
font-family: Avenir, "Avenir Next", Inter, Helvetica, Arial, sans-serif;
```

## UI Style

Visual direction:

- Clean.
- Modern.
- Technical.
- Sustainable.
- Minimal.
- Spacious.
- Trustworthy.
- Data-driven.

Avoid:

- Overly childish green visuals.
- Excessive leaf icons.
- Generic eco cliches.
- Dense enterprise clutter.
- Dark, heavy cyberpunk aesthetics.
- Unreadable gradients.

## Icon Style

Use simple line icons for:

- Cloud.
- Energy.
- Carbon.
- Security.
- Data.
- Compliance.
- Region.
- AI.
- Infrastructure.
- Dashboard.
- Workload optimization.

## Infographic Style

Use clear percentages, bars, dashboards, maps, and simple flow diagrams.

Preferred visuals:

- Carbon-aware workload routing diagram.
- Cloud emissions dashboard mockup.
- Region comparison map.
- ESG reporting flow.
- Tiered customer journey.
- GreenOps optimization loop.
