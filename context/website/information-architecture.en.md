# Website Information Architecture

This is a marketing-oriented information architecture option. It should be reconciled with `PROJECT_BRIEF.md` before implementation if route names conflict.

## Recommended Main Navigation

1. Platform.
2. Solutions.
3. Sustainability.
4. Security.
5. Resources.
6. Company.
7. Contact / Get Started.

## Platform Pages

- `/platform`: main platform overview.
- `/platform/self-service-cloud`: self-service cloud for startups, developers, and small teams.
- `/platform/private-cloud`: private cloud for enterprises and governments.
- `/platform/greenops-ai`: carbon-aware infrastructure and optimization for AI workloads.
- `/platform/wordpress-vps`: carbon-aware WordPress VPS.
- `/platform/carbon-aware-engine`: explanation of GREENPOW's carbon-aware logic, ranking, and workload optimization approach.

## Solutions Pages

- `/solutions/startups`: freelancers, founders, startups, and indie teams.

- `/solutions/government`: governments and public institutions.
- `/solutions/fintech`: fintech.
- `/solutions/ai`: AI companies.
- `/solutions/blockchain`: blockchain and distributed technology teams.
