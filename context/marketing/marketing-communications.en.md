# GREENPOW Marketing And Communications Context

## Purpose

This file gives developers, designers, copywriters, AI agents, and contributors the shared marketing and communications context required to build the GREENPOW web experience.

Use this context for:

- Website copy.
- Landing pages.
- Product pages.
- SEO pages.
- Campaign pages.
- Blog and editorial content.
- Sales enablement content.
- Localization.
- Brand voice.
- Persona-specific messaging.
- Tier and region-specific messaging.
- AI-assisted content generation.

GREENPOW should always be positioned as a trusted leader in carbon-aware cloud computing: sustainable, secure, privacy-first, scalable, measurable, and commercially practical.

## North Star Message

GREENPOW makes cloud infrastructure carbon-aware, secure, and scalable so organizations can grow digitally while reducing environmental impact and improving sustainability visibility.

## Category Ownership

GREENPOW should win by owning a sharper category:

```txt
Carbon-aware cloud.
```

Use this phrase consistently across the website, campaigns, and thought leadership.

## Core Strategic Reminder

GREENPOW should not try to win by saying it is a better generic cloud. The website, content, product pages, and campaigns should reinforce the carbon-aware cloud category consistently.

The goal is to be first in mind for carbon-aware cloud infrastructure across Europe and LATAM.
